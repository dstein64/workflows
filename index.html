workflows

Instructions

To view workflows for a user's public repositories, enter the corresponding user.

To view workflows for your public and private repositories, leave the user field blank, and enter an account token that has scope for accessing private repos.

When a token is specified, authenticated requests are made to GitHub's API, which permits a higher rate limit.

See GitHub's documentation for instructions on generating a token. No scopes are necessary to view workflows for public repositories. repo scope is necessary to view workflows for your private repositories.


connections The maximum number of concurrent connections for GitHub API calls.
default
branch Shows information for the latest workflow run on the default branch. Otherwise, information is shown for the latest workflow run on any branch.
